**Clone project** 

	git clone https://github.com/LeonardoWiest/wboss-server.git

**Build project**

	mvn clean install -Pdevelopment

(test,development,production)

**Run project**

	cd wboss-server/wboss-server-api/
	mvn spring-boot:run -Pdevelopment
